# Eclipse — Digital Afterlife Experience

A cinematic storytelling website exploring the idea of digital existence, memory preservation, and identity in the modern internet era.

This website was developed using only:

- HTML5
- CSS3
- JavaScript

The project is fully responsive and deployed using GitHub Pages as required in the task guidelines.

---

## Concept

The internet never truly forgets.

Every interaction, search, memory, and digital footprint becomes part of an invisible archive that continues to exist beyond human memory.

"Eclipse" visualizes this concept through cinematic layouts, immersive typography, animated interactions, atmospheric audio, and editorial-style storytelling.

---

## Features

- Cinematic hero section
- Custom animated cursor
- Interactive storytelling sections
- Timeline-based narrative
- Hover animations and transitions
- Ambient background audio
- Video integration
- Premium editorial footer
- Responsive design
- Fully static website
